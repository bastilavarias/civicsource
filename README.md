# Civic Source Front-end (Vue.js Setup)

## Requirements:
- Node.js
- Vue CLI

## To use this project in your machine, you need to run these following commands in your terminal.

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
